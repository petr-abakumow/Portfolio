Функционал, который я могу реализовать в проекте:

  • создать блоки с данными любого типа: текстовые, изображения, видеофайлы, видео со сторонних ресурсов, аудио, аналитика, карточки товаров и пр.
  • создать любые элементы интерактивности на странице: формы (а также настроить их валидацию), кнопки, чекбоксы, радиокнопки, модальные окна, карусели изображений
  • создать файловую структуру в проекте, которую легко и удобно будет поддерживать при любом расширении проекта
  • написанный код также будет понятен другим разработчикам и лёгок для поддержки
  • написать код, который будет работать в любых браузерах и корректно отображать интерфейс на любых устройтвах
  • реализовать функционал для пользовательского интерфейса: возможность изменять данные на странице, добавлять / удалять/ копировать / сортировать элементы (карточки товара, галереи изображений, плейлисты), ставить/снимать лайки. Например, для личного кабинета это нужно
  • реализовать анимацию, как средствами CSS, так и c помощью javaScript технологий
  • настроить функционал обработки запросов пользователя, в зависимости от его действий на странице
  • при работе с формами написать функционал для обработки, сохранения, изменения, сортировки, отправки на сервер данных, введённых пользователем в соответствующие поля. При неправильном или неполном заполнении полей кнопка сохранения неактивна, пользователь видит подсказку и рекомендации к исправлению, например если в поле где нужен номер телефона не хватает цифр и т.д.
